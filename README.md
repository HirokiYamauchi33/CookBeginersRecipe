# CookBeginersRecipe
php自作

##概要
料理レシピの投稿サイトを作成しました。
投稿の際に難易度を決めることができます。


##使い方
一般ユーザー
難易度別のレシピの表示、レシピの投稿と編集ができます。

メールアドレス→yamauchi@test.jp
パスワード→yamauchi

管理者ユーザー
一般ユーザーのできることに加えて、ユーザー一覧を表示とユーザーの削除、投稿一覧の表示と削除を行うことができます。
